# 🎓 Student Performance Prediction ML System

## Run Locally

pip install -r requirements.txt  
streamlit run app.py  

## Deploy

1. Upload to GitHub  
2. Go to Streamlit Cloud  
3. Select repo  
4. Deploy

- ML Prediction
- Mentor Advice
- Future Guidance
- Study Plan
- Visualization
